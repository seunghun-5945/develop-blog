## Step1. 우분투 설치하기

우선 시작하기에 앞서 노트북은 깔끔하게 포맷 해주세요 일반적인 노트북은 windows  
운영체제가 설치 되어 있을테니 os 까지 싸악 밀어버립시다~  
포맷을 완료했다면 노트북에 우분투를 설치하기 위해 부팅디스크(usb)를 만들어야 했는데요
저는 Rufus 라는 프로그램을 사용했습니다.  
우선 우분투 부터 다운로드 받읍시다.

[우분투 데스크탑버전 다운로드](https://ubuntu.com/download/desktop)  
[우분투 서버 버전 다운로드](https://ubuntu.com/download/server)

저는 안정화 버전인 LTS 버전 다운을 추천합니다.
[![Rufus 사이트](https://rufus.ie/pics/rufus-128.png)](https://rufus.ie/ko/)
우선 iso 형식의 우분투 파일을 받아봅시다. 참고로 저는 리눅스에 익숙하지 않기 때문에  
우분투 서버 버전을 받았다가 다시 gui 버전으로 재설치했답니다.
